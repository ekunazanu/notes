## Regression & Classification

* **Regression** — Predicting a value from given input value or values. Returns continuous values. If more than one value is returned, it is called **multivariate regression**.
* **Classification** — Predicting a category. Returns discrete categories. If prediction is *from* one of two categories, it is called **binary classification**. If prediction is *from* more than two categories, it is called **multiclass classification**.
